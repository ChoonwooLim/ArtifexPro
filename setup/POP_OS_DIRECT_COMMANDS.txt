【옵션 1: 한 번에 모두 실행】

echo "Jiyeon71391796!" | sudo -S bash -c '
apt update && apt install -y openssh-server

netstat -tlnp | grep :22

【옵션 2: 단계별 실행 (문제 파악용)】
--------------------------------------------
1. SSH 설치 확인:
   which sshd
   
   없으면 설치:
   sudo apt update
   sudo apt install openssh-server
   (비밀번호: Jiyeon71391796!)

2. SSH 서비스 상태 확인:
   sudo systemctl status ssh
   (비밀번호: Jiyeon71391796!)

3. SSH 시작:
   sudo systemctl stop ssh
   sudo systemctl start ssh
   (비밀번호: Jiyeon71391796!)

4. 방화벽 임시 비활성화:
   sudo ufw disable
   (비밀번호: Jiyeon71391796!)

5. SSH 포트 확인:
   sudo netstat -tlnp | grep :22
   또는
   sudo ss -tlnp | grep :22
   (비밀번호: Jiyeon71391796!)

6. IP 주소 확인:
   ip addr show enp1s0f1

=============================================

【옵션 3: 문제 진단】
--------------------------------------------
Pop!_OS에서 자체 테스트:
ssh stevenlim@localhost

로그 확인:
sudo journalctl -u ssh -n 50

=============================================

【중요】
--------------------------------------------
위 명령어 실행 후 다음이 보여야 합니다:
● ssh.service - OpenBSD Secure Shell server
     Loaded: loaded
     Active: active (running) ← 이것이 중요!

tcp    0    0 0.0.0.0:22    0.0.0.0:*    LISTEN    [숫자]/sshd
↑ 이것이 보여야 SSH가 실행 중입니다

=============================================

실행 완료 후 Windows에서:
ssh stevenlim@192.168.219.150
